FreeBSD/ppc Project

Introduction

The FreeBSD/ppc project pages contain information about the FreeBSD port to the PowerPC
architecture. As with the port itself, these pages are still a work in progress.

Status

The FreeBSD/ppc port is still a Tier 2 platform. Thats means it is not being fully supported by our security
officers, release engineers and toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC G5 has been commited to SVN.
  • 03 March, 2008: Support for Freescale PowerQUICC III MPC85XX faimly system-on-chip
    developments board has been commited into CVS.
  • 27 February, 2008: FreeBSD 7.0 is the firts release to officially support the FreeBSD/ppc port.
